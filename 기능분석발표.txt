1. 프로젝트의 개요 (주제): 주제를 택한 이유, 구체적으로
코로나로 인해 우울증 환자가 역대 최대 (기사, 데이터) -> 코로나 블루를 극복하기 위해 어떤게 행복과 연관이 있는지 궁금 ->
행복지수 예측 및 분석

2. 프로젝트 참여인원: 3명 
        담당파트: 데이터 수집 = 차재경, 분석 = 강민규 , 시각화(전처리), ppt = 강재연

3. 프로젝트 일정 -> 질문
       제안 (주제 설정) ->   분석   ->   설계    -> 디자인 -> 개발 및 테스트

4.    - 4. 상세 제안 내용(기능정의서)

         - 데이터 수집 : 전세계 행복지수 리포트 사이트 (https://worldhappiness.report/)
         - 데이터 저장 : csv
         - 데이터 처리 : 데이터의 전처리 
         - 시각화 : 각 기능별 필요 그래프
